# 这是一个自己写的 web 前端构建框架
## 使用到的包；查看 package.json 文件  
##  安装项目依赖
```
npm install
```
## 开发环境编译
```
npm run build:dev
```
## 生产环境编译
```
npm run build:prod
```
## 热启项目文件
```
supervisor ./build/app.js
```
## 浏览器访问项目
```
//开发环境
http://localhost:8081
//生产环境
http://localhost:80
```
